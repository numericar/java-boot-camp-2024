Variable (ตัวแปร)
    - ชื่อที่กำหนดให้กับข้อมูลที่เราต้องการจะเก็บ

Data Type (ชนิดตัวแปร)
    - ชนิดของข้อมูลที่ต้องการจะเก็บไว้ที่ตัวแปร
    - สามารถจะเก็บได้เฉพาะประเภทข้อมูลที่ระบุเอาไว้

Operators (ตัวดำเนินการกับตัวแปร)
    เป็นตัวที่ใช้ดำเนินการจัดการกับ data ให้สร้างผลลัพธ์ออกมาได้

* Data (ข้อมูล): เป็นข้อมูลดิบ ยังไม่เคยผ่านการประมวลผล อาจมีความหมายหรือไม่มีความหมายก็ได้ โดยจะมีอยู่ 2 แบบ คือ 
    Data: Qualitative 
        เป็นข้อมูลเชิงคุณภาพ เป็นข้อมูลที่ดูแล้วไม่สามารถแทนได้ด้วยตัวเลข เช่น เพศ
    Data: Quantitative Data
        เป็นข้อมูลเชิงปริมาณ สามารถสามารถแทนได้ด้วยตัวเลข เช่น อายุ รายได้

* Information (สารสาเทศ): เป็น Data ที่ผ่านการประมวลผลแล้ว โดยชุดข้อมูลจะต้องมีความหมาย

* Primitive Type: เป็นข้อมูลชนิดดั้งเดิมของภาษา เช่น byte, short, int, long, float, double, char, boolean โดยจะถูกเก็บไว้ใน stack เมื่อสร้างไว้ใน method
* Reference Type: ตัวแปรจะไม่เก้บค่าโดยตรง แต่จะจัดเก็บ address บน ram ที่อ้างอิงไปยังข้อมูล โดย address จะถูกจะเก็บไว้ใน stack และชื้อไปยังตำแหน่งของข้อมูลที่อยู่ใน heap

* memory stack: มีพื้นที่น้อยแต่ไว โดยตัวแปรที่ถูกสร้างไว้ใน method จะถูกเก็บไว้ที่นี่
* memory heap: มีพื้นที่เยอะ แต่การเข้าถึงช้าและมีความ dymamic

* signed integer: เก็บได้ทั้ง + และ -
