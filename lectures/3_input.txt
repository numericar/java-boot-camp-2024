Rules of Java program 
    1. Java จะเริ่มทำงานโปรแกรมที่ method main() เสมอ

* Hard code: เป็นการเรียกข้อมูลที่ถูกกำหนดค่าเข้าไปตรง ๆ เช่น int x = 10; ไม่มีการผ่านการประมวลผลมาก่อน (การดึงข้อมูลจากฐานข้อมูล เป็นต้น)

Scanner
    เป็นการรับชุดข้อมูลเข้ามาจาก input บนระบบ
    โดยจะมีฟังชันก์ให้ใช้งานดังนี้
        1. next() เป็นการอ่าน input อะไรก็ตามที่เราใส่เข้าไป
        2. nextLine() เป็นการอ่าน input และขึ้นบรรทัดใหม่

(System.in): เป็นการรับข้อมูลจาก console

* variable (ตัวแปร): Reference of value to use in the future, เป็นการที่เราเก็บข้อมูลไว้สักที่หนึงภายใน memory บนแรมเพื่อให้เราสามารถเรียกใชงานมันได้ในอนาคต

* ถ้าหากมีการใช้รับข้อมูลเป็นจำนวนมาก การใช้งาน InputStreamReader จะมีความเร็วมากกว่า System.in
