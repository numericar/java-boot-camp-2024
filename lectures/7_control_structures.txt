Control Structures
เป็นสิ่งที่ที่จะใช้วางแผนในการสั่งงานคอมพิวเตอร์ เพื่อประมวลผลข้อมูลจากจุดหนึ่งไปอีกจุดหนึ่ง
แล้วดูว่าสามารถเกิดผลลัพธ์อะไรได้บ้าง

link: https://wuttipong237.blogspot.com/p/control-structure.html

Sequence
    เป็นการทำงานตามลำดับ จากบนลงล่าง

Selection
    เป็นการตัดสินใจเลือก

Iteration
    เป็นการทำซ้ำขั้นตอนจนกว่าปัญหาจะถูกแก้ไข โดยจะมีการเปลี่ยนแปลง input ทุกครั้งที่ทำซ้ำ
    (แก้ปัญหาเดิมทุกการวนซ้ำ แต่ได้ผลลัพธ์ที่แตกต่าง)

Selection Techniques
    - simple if: ทำงานจากบนลงล่าง โดยอาจจะมีการประมวลผลแทรก
    - if-else:
    - if-else-if lodder
    - nested if: เป็นเงื่อนไขที่มีการซ้อนกัน โดยเมื่อผ่านเงื่อนไขแรก ก็จะมีอีกเงื่อนไขขึ้นมาเช็คอีกที

Iteration Techniques
    - for loop: รู้จำนวนรอบในการทำซ้ำ
    - while/do while: ทำซ้ำจนกว่าจะไม่เข้าเงื่อนไข (ข้อควรระวัง อาจเกิด infinite loop หรือ การทำซ้ำที่เป็นอนันต์)
    - recursive function calls: เรียกใช้งาน method/function ของตัวเอง โดยเมื่อผลลัพธ์ยังไม่ถูก resolve จะเรียกใช้งานตัวเองอีกครั้งโดยส่ง output ที่ได้จากการประมวลผล ณ ปัจจุบันเข้าไป
    - foreach loop
